This extension is for language learners and those who wish to hide quite a few distractions on YouTube. 

## How it works
- This is a Chrome extension for YouTube that filters results based on the selected language.
- The goal is to by-pass AI voiceover and AI-translated titles in YouTube to find videos that have original audio in a selected language.
- It's also possible to add a filter for the channel's country
- There are additional features to minimize distractions on YouTube by hiding view counts, upload date, like/dislike buttons, comments, etc. for a more focused experience on YouTube. 

## API
- This extension uses a YouTube Data API v3 key. This is necessary for accurate language filtering.
- You must create your own API key and paste it into the extention.

### How do I get an API key?
- Visit: console.cloud.google.com
- Follow the directions by Google: https://developers.google.com/youtube/v3/getting-started
